== Description ==

= Now with video support =

Aparg Slider is image and video(YouTube/Vimeo) slider with descriptions for each slide and smooth animations.
This plugin let users  create multiple sliders with descriptions for each slide.
You can create multiple sliders and embed slider in both template or post.
Each slide can have multiple descriptions with custom colors.
There are a lot of settings that can customize your slider.
Plugin is translation ready.

For demo visit <a href='http://wpdemo.aparg.com'>wpdemo.aparg.com</a>.

For support write to <a href='mailto:support@aparg.com'>support@aparg.com</a> and mention plugin name in subject.

For more info visit <a href='http://aparg.com'>aparg.com</a>.

== Installation ==

Automatic Installation:

1. Go to 'Admin' -> 'Plugins' -> 'Add New' and search for "Aparg Watermark and Resize"

2. Click on the 'Install Now' button

3. Click on the 'Activate Plugin' link


Manual Installation:

1. Download plugin archive

2. Go to 'Admin' -> 'Plugins' -> 'Add New' and click on the 'Upload Plugin' button

3. Choose plugin archive and click on the 'Install Now' button

4. Click on the 'Activate Plugin' link
